# jovial - a jovial theme for zsh

## Feature

- face score
- state
  - host and user
  - current path
  - development environment segment
  - git branch
  - time in line end
  - venv


## Install

if u are login as **root**, pls use this command:

```bash
curl -sSL git.io/jovial | bash
```

otherwise, use this below:

```bash
sudo bash -c "`curl -sSL git.io/jovial`" -- $USER
```
